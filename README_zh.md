## 说明

Tjujuk Tetris 是一款基于 Jetpack Compose 构建的安卓方块益智游戏。

<img src="app/src/main/res/drawable/ic_launcher.png" width=110 align=left hspace="10" vspace="10">

- 支持离线单机游玩
- 使用柔和红色主题界面
- 支持声音开关与暂停/继续
- 已准备好签名发布版本与 Android App Bundle

## 游戏规则

- 消除 1 行：100 分
- 消除 2 行：300 分
- 消除 3 行：700 分
- 消除 4 行：1500 分
- 随着消除行数增加，方块掉落速度会提升
